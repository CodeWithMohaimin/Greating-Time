# Greeting-Time

Just play with Javascript Dom manipulation. This project shows you whether this year is a Leap year or not based on your input year.

## Technologies Used

- HTML
- CSS
- JavaScript

## Features

- A responsive design that works well on different devices.
- A custom CSS layout.
- show greeting
